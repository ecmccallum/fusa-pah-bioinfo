Environment Setup

Funannotate was not available in the existing fusarium_env and was not found on the active PATH, so a dedicated environment was created using mamba rather than conda. Mamba uses a C++ dependency solver and is substantially faster for large dependency trees — funannotate has approximately 100 dependencies and conda's solver is prone to hanging on problems of this size. 

mamba create -n funannotate_env -c conda-forge -c bioconda funannotate
conda activate funannotate_env

The following environment variables are required and set permanently in ~/.bashrc:

export FUNANNOTATE_DB=/home/22/emccallum/funannotate_db
export GENEMARK_PATH=/home/22/emccallum/gmes_linux_64_4
export PATH=/home/22/emccallum/gmes_linux_64_4:$PATH

5.2 funannotate sort

The sort step renames contig headers from verbose SPAdes identifiers (e.g. NODE_1_length_5050286_cov_47.3) to short standardised identifiers (scaffold_1, scaffold_2, etc.) and sorts contigs by descending length. Many downstream tools cannot handle long or special-character contig names. A known bug in funannotate v1.8.17 causes a TypeError if --minlen is not explicitly passed; fixed with --minlen 0. 

funannotate sort \
  -i results/annotation/funannotate/fusarium.clean.fasta \
  -o results/annotation/funannotate/fusarium.sorted.fasta \
  -b scaffold \
  --minlen 0

Run interactively on login node. Completed in under one minute.

Sort verdict. 4,308 contigs sorted by length and renamed scaffold_1 … scaffold_4308.

5.3 funannotate mask — SLURM job 1572174

The mask step identifies repetitive regions (transposable elements, simple repeats) using tantan and soft-masks them with lowercase letters. Soft-masking marks repetitive regions in lowercase so downstream gene predictors can treat these regions differently and reduce false gene calls in repetitive sequence. 

funannotate mask \
  -i results/annotation/funannotate/fusarium.sorted.fasta \
  -o results/annotation/funannotate/fusarium.masked.fasta \
  --cpus 8

5.4 funannotate predict

The predict step is the core gene-finding stage. Funannotate runs multiple ab initio predictors — Augustus, GeneMark-ES, SNAP, GlimmerHMM, and CodingQuarry — trains gene models using BUSCO outgroups, and uses EvidenceModeler (EVM) to produce a consensus gene set weighted by predictor reliability. Gene models are assigned the locus tag prefix FUS.

Note on flags: -s sets the organism label written into output files. --augustus_species and --busco_seed_species independently specify which pre-trained models to use — F. graminearum is the closest available trained species in the Augustus database. --protein_evidence passes external protein sequences to EVM to provide additional support for evaluating gene models.

Comparison with F. solani SB1 reference annotation. The F. solani SB1 reference genome (GCF_023522795.1), used by Novogene as the mapping reference for resequencing, carries 18,900 annotated mRNA gene models in its NCBI RefSeq GFF3 annotation file. The v2 predict run produced 22,338 predicted protein-coding gene models — approximately 3,400 more than the reference strain, representing an 18% excess. A proportion of this excess is likely attributable to ab initio over-prediction in the absence of RNA-seq evidence, which is expected for this pipeline configuration. However, some fraction may reflect genuine genomic differences between the PAH-contaminated soil isolate and the SB1 reference strain, which is one of the biological questions this de novo assembly was designed to address. Distinguishing between these two sources will require functional annotation filtering in Stage 6 and discussion with the postdoc regarding the plausibility of the gene count given the assembly statistics.
BUSCO verdict. The genome BUSCO score of 98.3% complete indicates a near-complete assembly with respect to conserved fungal core genes. The protein BUSCO score of 97.0% complete with 0% duplication and 0% fragmentation indicates a high-quality predicted gene set. The 0% duplication result in protein mode argues against large-scale duplication of conserved genes, although it does not exclude smaller-scale overprediction of non-BUSCO ORFs. The total gene count of 22,338 is higher than the 12,000–16,000 range reported for many published F. solani genomes and warrants discussion with the postdoc; however, the BUSCO scores argue against a catastrophic annotation failure or major assembly duplication artefact.

Conclusions

Gene prediction and functional annotation are complete. The v2 predict run produced 22,338 predicted protein-coding gene models and 341 tRNAs from a 57.4 Mb assembly. BUSCO scores of 98.3% complete at genome level and 97.0% complete at protein level, with low duplication, indicate that the assembly and conserved gene space are highly complete. The 22.7% exon-protein evidence overlap represents a 7.5-fold improvement over v1 and indicates substantially improved protein support for the v2 gene models. Functional annotation added Pfam domain assignments to 15,170 genes, CAZyme annotations to 754 genes, protease annotations to 760 genes, and UniProt product names to 1,177 genes. eggNOG, InterProScan, SignalP, and Phobius were unavailable, so GO terms, KEGG-style orthology assignments, InterProScan domain annotations, and secretome predictions are absent. The total gene count remains higher than expected for many published Fusarium genomes and should be discussed with the postdoc, but the BUSCO results argue against major assembly duplication or a catastrophic annotation failure. The annotated gene set is now ready for candidate PAH-relevant enzyme screening in Stage 6.
